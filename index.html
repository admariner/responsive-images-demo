srcset with x-descriptors

You can alternatively give resolution info instead of width for each file, if the width is the same for all screen sizes. I usually find this more confusing. Leave out the resolution for the base low-resolution version since 1x is implied.

Art Direction with picture

Sometimes it's better if the image itself changes for different display sizes. "For example, if a large landscape shot with a person in the middle is shown on a website when viewed on a desktop browser, then shrunk down when the website is viewed on a mobile browser, it will look bad as the person will be really tiny and hard to see. It would probably be better to show a smaller, portrait image on mobile, which shows the person zoomed in." 1 It also allows us to use new image file types like webp while still providing fallback image types. See the source for more info.
